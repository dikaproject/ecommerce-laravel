Pertanyaan yang Sering Diajukan

Jawaban untuk pertanyaan umum tentang layanan kami

Berapa lama waktu pengerjaan untuk setiap layanan?

Waktu pengerjaan bervariasi tergantung pada kompleksitas proyek dan jenis layanan. Secara umum, untuk desain konten dan poster membutuhkan waktu 3-5 hari kerja, video konten 7-14 hari kerja, dan desain PPT 5-7 hari kerja. Kami akan memberikan estimasi yang lebih akurat setelah konsultasi.

Berapa kali revisi yang bisa saya minta?

Jumlah revisi tergantung pada paket yang Anda pilih. Paket basic biasanya mendapatkan 2 kali revisi, paket standard 3 kali revisi, dan paket premium 5 kali revisi. Revisi tambahan dapat dilakukan dengan biaya tambahan.

Apakah saya mendapatkan file source dari desain?

Ya, Anda akan mendapatkan file source (seperti PSD, AI, atau file project lainnya) untuk paket standard dan premium. Untuk paket basic, Anda akan mendapatkan file output final (seperti JPG, PNG, PDF, atau MP4).

Bagaimana cara melakukan pembayaran?

Kami menerima pembayaran melalui transfer bank dan QRIS. Untuk proyek dengan nilai di atas Rp 2.000.000, kami memerlukan down payment sebesar 50% di awal, dan pelunasan setelah proyek selesai sebelum file final dikirimkan.
